# Промпт для генерации массива новостей ПрофБазы

Ты помогаешь подготовить контент для учебного веб-портала "ПрофБаза". Это информационная база для молодых специалистов Республики Беларусь. Проект помогает с адаптацией на первом рабочем месте: трудоустройство, документы, медосмотр, правовые источники, новости по специализациям, инциденты на работе, организации с локальными документами. Первая пилотная специализация: "Педагогика".

Нужно собрать большой массив тем и фактуры для редакционного раздела "Новости и полезные материалы", будто у проекта есть редакция, которая несколько лет регулярно публикует полезные статьи.

Задача:
1. Подбери категории новостей, подходящие проекту. Например: трудоустройство, педагогика, право, охрана труда, адаптация, инциденты, документы организации, профессиональное развитие. Можешь предложить лучше.
2. Подготовь 80-120 идей публикаций за период 2023-2026 годов.
3. Для каждой публикации дай:
   - заголовок;
   - краткое описание на 1-2 предложения;
   - категорию;
   - специализацию: "Педагогика", "Право", "Информационные технологии", "Медицина", "Социальная работа" или "Общая";
   - аудиторию: "Новичок", "Опытный" или "Все";
   - дату публикации;
   - автора, реалистичное имя или "Редакция ПрофБазы";
   - 4-7 тегов;
   - 2-5 надежных источников;
   - короткий план статьи из 4-6 пунктов;
   - что можно показать гостю в открытом просмотре;
   - что оставить только для зарегистрированного пользователя.
4. Источники должны быть достоверными. Приоритет:
   - Национальный правовой интернет-портал Республики Беларусь: https://pravo.by/
   - Министерство образования Республики Беларусь: https://edu.gov.by/
   - Министерство труда и социальной защиты Республики Беларусь: https://mintrud.gov.by/
   - Министерство здравоохранения Республики Беларусь: https://minzdrav.gov.by/
   - Белстат: https://www.belstat.gov.by/
   - официальные сайты госорганов, учреждений образования, правовые справочные системы с указанием, что официальный текст нужно сверять на pravo.by.
5. Не выдумывай факты, номера актов и цитаты. Если точный номер документа не найден или не уверен, напиши "нужно проверить по официальному источнику".
6. Не перепечатывай большие куски нормативных актов. Дай краткое резюме и ссылку на источник.
7. Пиши для интерфейса: коротко, понятно, без рекламного тона. Материалы справочные, не юридическая консультация.
8. Отдельно выдели 20 материалов, которые лучше всего подходят для гостевой лендинговой страницы.
9. Отдельно выдели 20 материалов для персональной ленты молодого педагога.
10. Отдельно выдели 20 материалов для опытного педагога или администратора организации.

Формат ответа:
Сначала таблица категорий. Затем JSON-массив публикаций по схеме:

```json
[
  {
    "title": "Название публикации",
    "summary": "Краткое описание",
    "category": "Категория",
    "specialization": "Педагогика",
    "audience": "Новичок",
    "publishedAt": "2026-05-18",
    "author": "Редакция ПрофБазы",
    "tags": ["трудоустройство", "документы", "педагогика"],
    "sources": [
      {
        "label": "Название источника",
        "url": "https://..."
      }
    ],
    "outline": [
      "Пункт плана 1",
      "Пункт плана 2"
    ],
    "guestPreview": "Что можно показать гостю",
    "registeredOnly": "Что открыть после входа"
  }
]
```

После JSON дай краткий список тем, где обязательно нужна ручная проверка актуальности источников перед публикацией.
